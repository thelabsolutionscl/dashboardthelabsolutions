# Notas del proyecto
Las specs y decisiones de este proyecto viven en:
/Volumes/mac/Dashboard/
Revisa esa carpeta antes de tareas grandes, y anota ahí las decisiones
de arquitectura o diseño importantes.

# Registro de investigaciones
Cuando investigues algo no trivial (búsqueda web, revisar código para
entender un problema, comparar opciones, armar un plan), guarda un
resumen en:
/Volumes/mac/Dashboard/Investigaciones/AAAA-MM-DD-tema-corto.md
Contenido mínimo: la pregunta o tarea, qué encontraste (con fuentes o
archivos revisados), y la conclusión o recomendación. No lo hagas para
preguntas triviales de una línea.
